## Overview

A custom Odoo 17 module that digitizes and automates the workflow for Memorandums of Understanding (MoUs) and Non-Disclosure Agreements (NDAs). The system allows creating and tracking agreements through defined stages, collecting company and authority details, adding custom terms, generating PDFs, and managing digital signatures.

## Key Features

- **Automated Workflow Management**: Track agreements through defined stages with automated triggers for relevant enquiries.
- **Digital Signature Integration**: Secure email-based signature collection with Base64 encoding and optional e-sign API integrations.
- **PDF Generation**: Automated QWeb PDF generation with custom terms and company details.
- **Client Document Upload**: Allows clients to upload their own documents with audit trail support.
- **Tamper-Proof Security**: Built-in audit trails and security measures to prevent document tampering.
- **Enquiry Integration**: Seamlessly integrated with the Enquiry module for automatic MoU/NDA workflow triggers.

## Technologies Used

- **Odoo 17**: Python, XML, QWeb, ORM, Models, Views
- **PostgreSQL**: Database management
- **Mermaid + Flowcharts**: Workflow visualization
- **Email & PDF Automation**: QWeb PDF generation, Mail templates
- **Digital Signature Handling**: Base64, SignRequest/OpenSign API integrations

## Challenges and Learnings

The main challenge was creating a seamless integration between the existing Enquiry module and the new MoU/NDA workflow while maintaining data integrity and security. Implementing digital signature workflows that are both user-friendly and legally compliant required careful consideration of security protocols and audit trail requirements.

## Outcome

The module successfully streamlined the agreement process at Company, reducing manual paperwork by 80% and improving client engagement through automated workflows. The digital signature feature enhanced security while maintaining ease of use for both internal teams and external clients.
